# Website Application

Angular app for the frontend of [eddie-summers.com](https://eddie-summers.com).

## :running: Run Locally

`npm install`

`ng serve --open`

## :house: Build

`npm install`

`ng build --prod`

Deploy `./dist/` to your remote root.

**First time only**: Upload `./src/.htaccess` to your remote at `<root>/dist/website/` to enable routing.
